# Preset Catalog

Characters live in `characters/<slug>/`.
Objects live in `objects/<slug>/`.
Voice avatars live in `voice_avatars/<provider>/<voice_id>/`.

Each character/object folder can contain:

- `avatar.png`
- `reference_1.png`
- `reference_2.png`
- `reference_3.png`
- `prompt.txt`

Each voice avatar folder can contain:

- `avatar.png`
